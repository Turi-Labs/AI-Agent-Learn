# Contributing to AI Agent Learn

Thanks for wanting to contribute. This is a community-built resource and every improvement — big or small — matters.

## What you can contribute

- **Fix errors** — typos, broken links, outdated information
- **Improve explanations** — clearer wording, better examples, diagrams
- **Add code examples** — working code that illustrates a concept in a module
- **Write lessons** — new lessons within an existing module (check ROADMAP.md first)
- **Translate** — make the content accessible in other languages
- **Suggest topics** — open a Discussion if you want to see something covered

## How to contribute

1. **Fork** the repo and create a branch from `main`
2. Make your changes
3. Open a **Pull Request** with a clear description of what you changed and why
4. A maintainer will review and merge

For larger changes (new modules, structural rewrites), open a **Discussion** first so we can align before you spend time writing.

## Repo structure

```

│   ├── 01-ai-agent-fundamentals/

│   ├── 03-operators-and-principals/
│   └── 04-evals-and-benchmarks/
├── resources/             # Papers, PDFs, reference material
├── README.md
├── ROADMAP.md
└── CONTRIBUTING.md
```

## Writing style

- Write for someone encountering these concepts for the first time
- Prefer concrete examples over abstract definitions
- Keep sentences short
- Code examples should be minimal and runnable

## Code of conduct

Be kind. This is an educational space. Constructive feedback only.

If you have questions, open a [Discussion](https://github.com/Turi-Labs/AI-Agent-Learn/discussions).
